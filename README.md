# DjangoSimpleCMS

```
Tutorial from: www.youtube.com/@veryacademy

Goals:
 - Basic Content Management System with CRUD database actions
 - practising CBV (Class based views: ListView, DetailView, UpdateView, DeleteView, CreateView)
 - utilizing widgets (django-widget-tweaks package - help for managing / formating forms)
 - showing how easy is to build CRUD system using builtin generic views

Next steps:
 - adding protection of adding/updating/deleting of posts (logging, authentication, authorization/permissions)
```
